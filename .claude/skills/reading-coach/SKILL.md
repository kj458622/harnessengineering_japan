# 히라가나 읽기 코치 스킬

일본어 단어의 발음을 한국 학습자가 빠르게 익힐 수 있도록 음절 단위 분석과 기억법을 제공한다.

## 분석 절차

### 1. 읽기 확인

- 단어의 표준 히라가나 표기를 확인한다
- 로마자(Romaji) 표기도 함께 제시한다
- 읽기가 여럿인 경우(음독/훈독): 이 단어에서 쓰이는 읽기를 우선하고, 다른 읽기는 참고로 언급한다

### 2. 음절 분해 및 기억법

각 음절(モーラ)에 대해 아래 4가지를 제공한다:

**① 로마자 표기**
- Hepburn 표기법 기준

**② 한국어 유사음**
- 가장 가까운 한국어 발음을 찾는다
- 완전히 같지 않으면 차이를 명시한다
- 예: "ふ(fu)는 한국어 '후'와 비슷하지만 입술을 붙이지 않고 윗니를 아랫입술에 살짝 댄다"

**③ 글자 모양 연상법**
히라가나 글자의 시각적 특징을 활용해 발음과 연결하는 이미지를 만든다.

잘 알려진 연상법 예시:
- あ (a): "아!" 하고 놀라는 사람
- き (ki): 나무(木)처럼 생겼다 → 나무는 'ki'(木)
- さ (sa): 한자 '차(且)'에서 왔다 → "サッ 사~"
- て (te): 알파벳 'て'처럼 생겼다 → 손(hand) 모양 → 'te'
- の (no): '의'를 뜻하는 の → 소용돌이처럼 "노~" 감기는 느낌

이미지가 억지스러우면 생략하고 반복 쓰기 권유로 대체한다.

**④ 발음 주의사항**
한국 학습자가 자주 틀리는 발음:
- っ(촉음): 다음 자음을 된소리로, 앞에서 잠깐 멈추는 느낌
- ん(발침): 뒤에 오는 자음에 따라 'n/m/ng' 발음이 달라진다
- 장음(ー/oo/uu): 한 박자 더 길게 늘린다
- は/へ/を: 조사로 쓰일 때 발음이 달라진다

### 3. 발음 리듬

단어 전체를 한국어 리듬감으로 표현한다.

**표현 방식:**
- 박자 표시: 몇 박자 단어인지 (예: "4박자")
- 억양: 일본어는 기본적으로 평탄하게 읽는다고 안내
- 리듬 표현: 한국어 단어나 문장으로 비슷한 리듬을 만든다
  - 예: "たべもの(ta-be-mo-no)" → "'타베모노' — 타코야키 먹으러 모노레일 타자!"

### 4. 실전 예문

학습한 단어가 들어간 간단한 일본어 문장 1개를 제시한다.

**예문 선택 기준:**
- 초급~중급 수준의 문법
- 상황이 명확해서 단어 사용 맥락을 이해할 수 있는 것
- 히라가나 병기 포함

**형식:**
```
예문: [일본어]
읽기: [히라가나]
뜻:   [한국어 번역]
```

## 출력 품질 기준

- 발음 정보는 표준 일본어(共通語) 기준
- 억지스러운 연상법은 쓰지 않는다 — 없으면 "익숙해지면 자연히 외워집니다"로 마무리
- 표 형식은 가독성을 위해 4열(글자/로마자/유사음/기억법)로 통일
