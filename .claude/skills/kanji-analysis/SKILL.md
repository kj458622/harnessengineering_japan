# 한자 분석 스킬

일본어 단어에 포함된 한자를 해부하여 한국 학습자가 직관적으로 이해하고 기억할 수 있도록 설명한다.

## 분석 절차

### 1. 단어 파악
- 입력된 단어에 포함된 한자 목록을 추출한다
- 각 한자의 일본어 음독(音読み)과 훈독(訓読み)을 확인한다
- 이 단어에서 어떤 읽기가 적용됐는지 명시한다

### 2. 부수 분해

각 한자를 구성하는 핵심 부수를 추출한다. 부수 설명 시 포함할 요소:

| 요소 | 설명 |
|------|------|
| 부수 이름 | 한자 이름 + 한국 음 (예: 木 나무 목) |
| 원래 의미 | 이 부수가 처음에 나타낸 것 (상형/지사/회의 등) |
| 시각적 원형 | 어떤 모양에서 왔는지 묘사 |
| 현재 용도 | 부수로 쓰일 때 어떤 의미 범주를 담당하는지 |

**부수 분해 깊이 기준:**
- 초등 한자(교육 한자 1006자): 2~3개 부수로 분해
- 중급 한자: 핵심 부수 2개 + 나머지 설명
- 복잡한 한자: 부수 나열보다 시각적 덩어리(塊)로 묶어 설명

### 3. 성립 원리 설명

부수들이 어떻게 합쳐져 현재 의미가 됐는지 설명한다.

**좋은 설명의 조건:**
- 논리적 연결: "A와 B가 합쳐져 C가 됐다" 식으로 납득이 가야 한다
- 역사적 근거: 가능하면 실제 어원에 기반한다. 민간어원(folklore etymology)임을 알면 "(학습용 연상입니다)"라고 표시한다
- 간결성: 3문장 이내

### 4. 암기 스토리 생성

한 문장짜리 생생한 스토리로 한자 전체를 기억하게 한다.

**스토리 작성 원칙:**
- 부수의 의미를 등장인물/소품으로 활용한다
- 결말이 한자의 의미와 연결돼야 한다
- 유머나 과장이 있으면 기억에 더 잘 남는다
- 한국 학습자에게 친숙한 상황/비유를 쓴다

**예시:**
- 明(밝을 명): "해(日)와 달(月)이 함께 떠 있으니 세상이 밝다(明)"
- 森(숲 삼): "나무(木) 세 그루가 모이면 숲(森)이 된다"

### 5. 단어 전체 통합

단어를 구성하는 한자들의 의미가 어떻게 합쳐져 단어 전체 의미가 됐는지 한 줄로 마무리한다.

## 출력 품질 기준

- 부수 분해가 틀리지 않아야 한다 (표준 214 부수 기준)
- 스토리가 너무 억지스러우면 생략하고 시각 연상으로 대체한다
- 한 한자당 설명이 5줄을 넘지 않도록 간결하게 유지한다
