# 경선식 스타일 일본어 단어 암기 앱 기획서

## 1. 제품 개요

이 앱은 일본어 단어를 한국어 사용자가 빠르게 외울 수 있도록 돕는 단어 학습 앱이다. 핵심은 일본어 발음을 한국어 말장난, 상황 이미지, 과장된 장면과 연결하는 **경선식 영단어식 암기 알고리즘**이다.

일본어는 영어와 달리 한자 의미가 중요하므로, 단순 소리 암기만 제공하지 않는다. 앱은 먼저 발음과 뜻을 연결하는 암기문을 제공하고, 그 다음 한자 의미, 부수, 예문, 발음 주의점을 보강한다.

## 2. 핵심 컨셉

**소리 하나 + 뜻 하나 + 장면 하나**

사용자가 단어를 보면 아래 순서로 기억하게 만든다.

1. 일본어 단어의 뜻을 쉬운 한국어로 이해한다.
2. 일본어 발음을 한국어처럼 들리는 소리로 잡는다.
3. 그 소리와 뜻을 한 문장짜리 장면으로 연결한다.
4. 한자가 있으면 한자 의미로 기억을 보강한다.
5. 짧은 예문으로 실제 쓰임을 확인한다.

예시:

```markdown
森（もり） — 숲

경선식 암기: 나무가 모리모리 모여 있는 곳이 숲.

한자 보강: 木가 세 개 모이면 森, 나무가 빽빽한 숲.
```

```markdown
暑い（あつい） — 덥다

경선식 암기: 아, 뜨시다! 너무 더워서 あつい.

핵심만: 끝의 い는 형용사 꼬리라 따로 외우지 않아도 된다.
```

## 3. 타깃 사용자

- 일본어를 처음 시작한 한국어 사용자
- 히라가나/가타카나를 외웠지만 단어가 잘 안 외워지는 학습자
- JLPT N5~N3 단어를 빠르게 외우고 싶은 사용자
- 한자 때문에 일본어 단어 암기에 막히는 사용자
- 기존 단어장 앱이 딱딱하고 재미없다고 느끼는 사용자

## 4. 사용자 문제

현재 일본어 단어 학습에서 사용자가 겪는 문제는 다음과 같다.

- 일본어 발음과 한국어 뜻이 따로 놀아서 오래 기억되지 않는다.
- 단어장이 단순 반복 위주라 지루하다.
- 한자 단어는 뜻이 왜 그렇게 되는지 이해하기 어렵다.
- 히라가나를 글자별로 외우면 실제 단어 암기에 잘 연결되지 않는다.
- 형용사 끝 `い`, 동사 어미 같은 문법 꼬리까지 억지로 외우다가 핵심을 놓친다.

## 5. 제품 해결 방식

앱은 단어를 아래 4단계 카드로 제공한다.

### 5.1 쉬운 뜻

단어의 핵심 뜻과 느낌을 초보자용 한국어로 설명한다.

예:

```text
都会는 시골이 아니라 사람이 많고 건물도 많은 번화한 도시를 말해요.
```

### 5.2 경선식 암기

발음과 뜻을 한 문장 장면으로 연결한다.

규칙:

- 전체 발음을 기준으로 만든다.
- 글자 하나하나를 억지로 쪼개지 않는다.
- 1문장 기본, 길어도 2문장.
- 재미보다 기억성을 우선한다.
- 억지 연결이면 소리 암기 대신 한자 의미 암기를 권한다.

예:

```text
とかい: “또 가이?” 사람들이 또 가는 곳, 사람 몰리는 곳이 都会 도시.
```

### 5.3 한자 보강

한자가 있는 단어는 부수와 의미 연결을 짧게 설명한다.

예:

```text
都 = 큰 도시
会 = 사람들이 모임
都会 = 사람들이 많이 모인 큰 도시
```

### 5.4 실제 예문

짧은 예문으로 단어의 쓰임을 확인한다.

예:

```text
東京は都会です。
도쿄는 도시입니다.
```

## 6. 핵심 알고리즘

### 6.1 단어 분석

입력 단어를 분석한다.

- 표기: 한자, 히라가나, 가타카나
- 읽기: 히라가나
- 로마자: Hepburn 기준
- 품사: 명사, 형용사, 동사 등
- 뜻: 한국어 핵심 뜻
- 한자 포함 여부
- 문법 꼬리 여부

### 6.2 소리 덩어리 추출

단어 전체 발음을 한국어 유사음으로 변환한다.

예:

- `もり` → 모리
- `ちず` → 치즈
- `あつい` → 아츠이
- `とかい` → 토카이

이때 글자별 암기법을 만들지 않고, 전체 소리 또는 의미 있는 소리 덩어리만 사용한다.

### 6.3 한국어 연상 후보 생성

한국어 유사음으로부터 연상 후보를 만든다.

예:

- `もり` → 모리모리
- `ちず` → 치즈
- `あつい` → 아, 뜨시다
- `とかい` → 또 가이?

### 6.4 뜻 연결 평가

연상 후보가 실제 뜻과 연결되는지 평가한다.

좋은 암기문 조건:

- 발음을 들으면 암기문이 떠오른다.
- 암기문을 보면 뜻이 떠오른다.
- 1초 안에 이미지가 그려진다.
- 문장이 짧다.
- 단어의 품사나 문법 꼬리를 왜곡하지 않는다.

나쁜 암기문 조건:

- 글자를 억지로 하나씩 해석한다.
- 뜻과 상관없는 말장난만 있다.
- 장면이 너무 길다.
- `い`, `る`, `ます` 같은 문법 부분을 핵심처럼 외우게 만든다.

### 6.5 한자 의미 보강

한자가 있으면 소리 암기 후 의미를 보강한다.

원칙:

- 한자 설명은 짧게.
- 초보자용 단어는 1~2줄로 충분.
- 부수는 꼭 필요한 경우만 표로 제공.
- 실제 어원이 불확실하면 “학습용 연상”으로 표시.

### 6.6 복습 스케줄링

사용자의 기억 상태에 따라 복습 시점을 조정한다.

기본 단계:

- 처음 학습
- 10분 뒤
- 1일 뒤
- 3일 뒤
- 7일 뒤
- 14일 뒤
- 30일 뒤

사용자가 “헷갈림”을 누르면 다음 복습 간격을 줄인다.

## 7. 주요 기능

### 7.1 단어 검색

사용자가 일본어 단어, 한국어 뜻, 발음 중 하나로 검색한다.

예:

- `森`
- `もり`
- `모리`
- `숲`

검색 결과는 단어 카드로 제공한다.

### 7.2 AI 암기 카드 생성

사용자가 단어를 입력하면 AI가 아래 항목을 생성한다.

- 쉬운 뜻
- 경선식 암기문
- 소리 느낌
- 핵심 발음 포인트
- 한자 의미 보강
- 예문
- 한 줄 요약

### 7.3 내 단어장

사용자는 마음에 드는 단어 카드를 저장한다.

필드:

- 단어
- 읽기
- 뜻
- 암기문
- 예문
- 난이도
- 마지막 복습일
- 다음 복습일
- 기억 상태

### 7.4 암기문 수정

사용자가 AI 암기문을 직접 수정할 수 있다.

이 기능이 중요하다. 경선식 스타일 암기는 개인에게 잘 맞는 말장난이 다를 수 있기 때문이다.

### 7.5 복습 모드

카드 앞면:

```text
都会（とかい）
경선식 힌트: “또 가이?”
```

사용자가 뜻을 떠올린 뒤 카드를 뒤집는다.

카드 뒷면:

```text
뜻: 도시, 도회지
암기: “또 가이?” 사람들이 또 가는 곳, 사람 몰리는 곳이 都会 도시.
예문: 東京は都会です。
```

버튼:

- 기억남
- 헷갈림
- 다시 보기

### 7.6 퀴즈 모드

문제 유형:

- 일본어 → 한국어 뜻
- 한국어 뜻 → 일본어
- 암기문 빈칸 채우기
- 예문 속 단어 맞히기
- 한자 의미 조합 맞히기

### 7.7 JLPT 덱

기본 단어장을 제공한다.

- N5
- N4
- N3

MVP에서는 N5만 제공한다.

## 11. AI 프롬프트 설계

AI는 단어 카드 생성 시 아래 규칙을 따른다.

```text
너는 한국어 사용자를 위한 일본어 단어 암기 코치다.
경선식 영단어 스타일로 일본어 전체 발음과 한국어 뜻을 연결하는 짧은 암기문을 만든다.

규칙:
- 글자별로 억지 분해하지 않는다.
- 전체 발음과 뜻을 하나의 장면으로 묶는다.
- 암기문은 1문장 기본, 길어도 2문장.
- 형용사 끝 い, 동사 어미 등 문법 꼬리는 핵심 암기법으로 만들지 않는다.
- 한자가 있으면 암기문 뒤에 한자 의미를 짧게 보강한다.
- 불확실한 어원은 확정적으로 말하지 않는다.
```

출력 JSON 예:

```json
{
  "expression": "森",
  "reading": "もり",
  "romaji": "mori",
  "meaningKo": "숲",
  "mnemonic": "나무가 모리모리 모여 있는 곳이 숲.",
  "easyExplanation": "森는 나무가 많이 모여 있는 숲이에요.",
  "kanjiNotes": [
    {
      "kanji": "森",
      "koreanSound": "삼",
      "meaning": "숲",
      "easyStory": "木가 세 개 모이면 나무가 빽빽한 숲이 됩니다."
    }
  ],
  "pronunciationNote": "짧게 모-리 2박자로 읽습니다.",
  "exampleJa": "森に行きます。",
  "exampleReading": "もりに いきます。",
  "exampleKo": "숲에 갑니다."
}
```

## 12. 품질 기준

좋은 카드:

- 뜻이 바로 이해된다.
- 암기문이 짧다.
- 발음과 뜻이 동시에 떠오른다.
- 한자 설명이 부담스럽지 않다.
- 예문이 쉽고 자연스럽다.

나쁜 카드:

- 글자별로 쪼개서 기억법을 붙인다.
- 말장난은 있는데 뜻이 안 떠오른다.
- 설명이 너무 길다.
- 한자 어원을 확정적으로 틀리게 말한다.
- 형용사/동사 어미까지 억지로 암기시킨다.
