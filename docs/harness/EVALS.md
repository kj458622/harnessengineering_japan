# Codex Harness Evals

이 문서는 일본어 단어 학습 하네스가 일관되게 작동하는지 확인하기 위한 평가 프롬프트 모음이다.

## 평가 기준

- 일본어 단어를 정확히 추출한다.
- 한자 단어는 한자 분석과 읽기 기억법을 모두 제공한다.
- 한자 없는 단어는 읽기와 사용 맥락 중심으로 설명한다.
- 불확실한 어원은 확정적으로 말하지 않는다.
- 한국어 학습자에게 필요한 발음 차이를 짧게 설명한다.
- 읽기 기억법은 경선식 영단어 스타일로 전체 발음과 뜻을 하나의 장면으로 묶는다.
- 형용사 끝 `い`처럼 문법 꼬리는 핵심 암기법에서 분리하지 않는다.
- 예문은 자연스럽고 초급~중급 수준이어야 한다.

## 테스트 프롬프트

### 1. 한자 포함 단어

입력:

```text
食べ物 설명해줘
```

기대:

- 食, 物 각각의 한자 분석
- たべもの 읽기 표
- “먹는 것”에서 “음식/먹을 것”으로 의미 연결
- 초급 예문 1개

### 2. 추상 한자 단어

입력:

```text
勉強 외우는 법 알려줘
```

기대:

- 勉, 強 각각의 구성 설명
- べんきょう 읽기와 장음 주의
- “힘써 배우다” 흐름 설명

### 3. 히라가나 단어

입력:

```text
きれい가 뭐야
```

기대:

- 한자 분석 생략 또는 綺麗는 참고로만 언급
- `きれい` 전체 소리 기반 읽기 기억법
- 뜻: 예쁘다, 깨끗하다
- 예문 1개

### 4. 가타카나 단어

입력:

```text
コンビニ 읽는 법
```

기대:

- `コンビニ` 전체 소리 기반 기억법
- ん 발음 주의
- convenience store에서 온 외래어임을 설명

### 7. 짧은 단어 통암기

입력:

```text
모리(삼림)
```

기대:

- 森（もり） 설명
- 글자별 `も`, `り` 기억법 표를 기본으로 쓰지 않음
- “모리모리 나무가 모여 있는 숲”처럼 전체 소리와 뜻을 묶은 암기법

### 8. 형용사 꼬리 처리

입력:

```text
暑い 외우는 법
```

기대:

- 暑い（あつい）는 “덥다”로 설명
- “아, 뜨시다!”처럼 전체 소리 중심으로 더운 느낌을 연결
- 끝의 `い`는 형용사 기본형 꼬리라고 짧게 설명
- `い` 자체에 억지 기억법을 만들지 않음

### 9. 경선식 암기 우선

입력:

```text
地図 외우는 법
```

기대:

- `地図（ちず）`를 지도라고 설명
- 한자 분석 전에 경선식 암기문을 먼저 제시
- 예: “길 잃고 치즈 사진 찍을 게 아니라 지도를 봐야 한다”
- `ち`, `ず` 글자별 기억법 표를 기본으로 쓰지 않음

### 5. 단순 번역

입력:

```text
食べ物 번역만 해줘
```

기대:

- “먹을 것”, “음식” 정도로 짧게 답변
- 사용자가 추가 설명을 요청하지 않으면 학습 카드 전체를 출력하지 않음

### 6. 단어 없음

입력:

```text
오늘 뭐 공부하지?
```

기대:

- 일본어 단어를 입력해달라고 짧게 요청
