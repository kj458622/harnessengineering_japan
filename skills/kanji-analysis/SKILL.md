# 한자 분석 스킬

일본어 단어에 포함된 한자를 해부하여 한국 학습자가 의미와 모양을 함께 기억하도록 돕는다.

분석은 쉬운 설명이 먼저다. 한자를 구성요소로 쪼개기 전에 “이 한자는 그림으로 보면 이런 장면”이라는 식으로 감을 잡게 한다.

## 분석 절차

1. 단어 안의 한자를 추출한다.
2. 각 한자의 한국 한자음, 일본어 음독/훈독, 이 단어에서 쓰인 읽기를 확인한다.
3. 핵심 부수와 구성요소를 2~3개 수준으로 나눈다.
4. 각 구성요소의 원래 의미, 시각적 모양, 현재 역할을 설명한다.
5. 전체 의미가 어떻게 만들어졌는지 1~3문장으로 연결한다.
6. 마지막에 한 문장짜리 암기 스토리를 제시한다.

## 출력 기준

```markdown
### 한자 분석

**먼저 쉽게:** {이 한자를 그림이나 상황으로 비유한 쉬운 설명}

**{한자}** ({한국 한자음}, {읽기}) — {의미}

| 구성 | 원형/의미 | 역할 |
|------|-----------|------|
| {구성요소} | {원래 모양 또는 의미} | {현재 한자에서의 의미 역할} |

- **성립 원리:** {간결한 연결 설명}
- **시각 연상:** {그림처럼 떠올릴 설명}
- **암기 스토리:** {한 문장 기억법}
```

## 판단 규칙

- 부수 표를 보여주기 전에 의미를 쉽게 풀어쓴다.
- 한자 초보자가 모를 만한 용어는 괄호로 설명한다.
- 교육 한자는 2~3개 구성요소로 분해한다.
- 복잡한 한자는 모든 획을 쪼개지 말고 시각적 덩어리로 묶는다.
- 실제 어원과 학습용 연상이 다르면 구분한다.
- 민간어원이나 창작 기억법은 “학습용 연상”이라고 표시한다.
- 희귀 한자는 “일반 학습 빈도는 낮은 한자”라고 알려주고 가능한 범위에서 설명한다.
