# 읽기 코치 스킬

일본어 단어의 소리와 뜻을 한국어 학습자가 빠르게 연결하도록 돕는다.

읽기 설명은 전체 소리의 느낌을 먼저 잡아준다. 글자별 표는 기본값이 아니다. 전체 발음과 뜻을 한 장면으로 묶는 경선식 영단어 스타일 암기법을 우선한다.

## 분석 절차

1. 표준 히라가나 또는 가타카나 표기를 확인한다.
2. Hepburn식 로마자를 함께 제시한다.
3. 전체 단어가 한국어로 어떻게 들리는지 먼저 적는다.
4. 전체 발음과 뜻을 하나의 장면으로 묶은 경선식 암기 문장을 만든다.
5. 촉음, 장음, ん, 조사 は/へ/を, 형용사 끝 い 등 꼭 필요한 발음/문법 포인트만 설명한다.
6. 실전 예문 1개를 제시한다.

## 출력 기준

```markdown
### 읽기 기억법

**먼저 소리로:** {한국어로 가장 가까운 소리와 전체 리듬}

**{단어}** → **{읽기}** ({로마자})

- **통암기:** {전체 발음과 뜻을 묶은 경선식 스타일 암기 문장}
- **핵심만:** {꼭 필요한 발음/문법 포인트만 1줄}

{사용자가 요청하거나 발음이 복잡할 때만}

| 덩어리 | 소리 | 포인트 |
|--------|------|--------|
| {소리 덩어리} | {한국어 유사음} | {주의점} |

### 예문

> {일본어 문장}
> {히라가나}
> {한국어 뜻}
```

## 품질 기준

- 표를 보여주기 전에 전체 단어가 어떻게 들리는지 쉬운 한국어로 먼저 말한다.
- 기본 응답에서는 글자 하나하나에 억지 기억법을 붙이지 않는다.
- `もり`처럼 짧은 단어는 “모리모리 나무가 모여 있는 숲”처럼 전체 소리로 외우게 한다.
- `暑い(あつい)`처럼 형용사 끝 `い`는 따로 암기법을 만들지 않는다. “끝의 い는 형용사 꼬리” 정도로만 설명한다.
- 경선식 암기문은 1문장으로 끝내는 것을 기본으로 한다.
- 발음과 뜻이 잘 붙지 않으면 억지 말장난보다 사용 장면이나 한자 의미를 우선한다.
- 장음은 “한 박자 길게 끄는 소리”, 촉음은 “잠깐 멈췄다가 다음 소리를 세게 내는 느낌”처럼 풀어쓴다.
- 표준 일본어(共通語)를 기준으로 한다.
- 한국어 유사음은 근사치임을 전제로 설명한다.
- 억지스러운 글자 연상은 만들지 말고 반복 읽기나 단어 속 위치 기억법으로 대체한다.
- 예문은 짧고 실제 사용 맥락이 분명해야 한다.
